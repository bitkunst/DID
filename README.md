## 📌 Project 소개

<br/>

### **Block-Chain 8 Weeks Cooperation Project**

<br>

Layer2 Optimism Contract에 사용자의 정보를 저장하는 DID 서비스와
DID계정을 이용해 타사이트에서 회원가입 및 로그인이 가능한 SSO 구현.
같은 DID 계정으로 인증한 사이트간 포인트공유.
